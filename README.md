# click-add-1-server
### 经典游戏“点我加一”的双人对战版，笔者增加了几个技能，提高游戏可玩性~
1. **文件使用方法：**
- PvP_deno_server.ts 使用[deno playground](https://dash.deno.com/account/overview)直接部署，支持10个房间对战，支持显示每个房间有几个人
- 本地双人对战.html 直接使用浏览器打开即可，支持双人对战
2. **Demo（对局密码全都是2025）**
